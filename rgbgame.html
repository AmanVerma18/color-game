The Great

color guessing game
CHANCES LEFT : 3

  • How to play
  • R, G, B corresponds to red, green and blue respectively.
  • Each of which has value between 0 and 255(both values inclusive)
  • Try to identify the color obtained by mixture of RED GREEN BLUE
  • You will get 3 chances in HARD mode to choose from 6 different colors
  • You will get 1 chance in EASY mode to choose from 3 different colors
  • Press the NEW COLORS button if you want to change colors.
  • Press TRY AGAIN? button if you want to try again.
